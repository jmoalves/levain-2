# Installation Options Guide

This document describes the supported installation path for Levain 2.

## JAR Distribution (Supported)

Levain 2 is distributed as a standalone JAR and requires Java 25 or later.

### Prerequisites
- Java 25+

### Build

Build instructions are consolidated in [docs/build/BUILD_RELEASES.md](docs/build/BUILD_RELEASES.md).

### Run

```bash
java -jar target/levain.jar [command] [options]

# Examples
java -jar target/levain.jar list
java -jar target/levain.jar install git
java -jar target/levain.jar shell jdk-21
```

### Bootstrap Installers

Release assets also include helper installers:
- `install-linux.sh`
- `install-macos.sh`
- `install-windows.ps1`

These installers download `levain.jar` and create a launcher command.

## Recommended Approach

1. Development: use local Maven build (`mvn clean package`)
2. CI/CD: use JAR execution for tests and automation
3. Distribution: publish `levain.jar` and installer scripts

## Testing Your Build

```bash
java -jar target/levain.jar --version
java -jar target/levain.jar list
```

## Troubleshooting

- `UnsupportedClassVersionError`: Java is too old, use Java 25+
- `NoClassDefFoundError`: use `target/levain.jar` generated by the project build
